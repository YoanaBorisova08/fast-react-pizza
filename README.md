## Fast React Pizza — Order Your Favorite Pizza Online 🍕

A modern pizza ordering web app where users can browse a menu, add pizzas to a cart, and place orders through a smooth and responsive interface. Built with React using component-based architecture and modern state management patterns.

## Features

🍕 Browse a dynamic pizza menu  
🛒 Add and remove items from the cart  
📦 Update item quantities in real time  
💰 Automatic total price calculation  
🚚 Simple and fast checkout flow  
⚡ Fast, responsive UI  
🧠 Clean state management with React hooks and reducers  
💾 Persistent cart state (if enabled)

---

## How to Use

1. Open the app and browse available pizzas
2. Add pizzas to your cart
3. Adjust quantities as needed
4. Review total price in the cart
5. Proceed to checkout and place your order

---

## Tech Stack

- React 
- JavaScript (ES6+)
- Tailwind
- Vite 
- Redux

## Running Locally

```bash
npm install
npm run dev   # start development server
npm run build # production build
```
